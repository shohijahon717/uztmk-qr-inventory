# UZTMK QR Inventory System

UZTMK uchun qurilmalarni inventarizatsiya qilish va QR kod orqali boshqarish tizimi.

## Imkoniyatlar

- Qurilma qo‘shish
- Inventar raqamni avtomatik yaratish
- QR kod generatsiya qilish
- Qurilmani xodimga topshirish
- Qurilmani qaytarish
- QR kod orqali qurilma ma’lumotini ko‘rish
- Excel eksport
- QR print sahifasi
- Admin login tizimi

## Texnologiyalar

- Python
- Django
- Django REST Framework
- Bootstrap
- SQLite

## Ishga tushirish

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
